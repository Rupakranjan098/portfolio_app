# Personal Portfolio

This is a Laravel-based personal portfolio website, featuring projects, services, and a contact section.

## Features
- Dynamic Project Showcase
- Services Section
- Admin Panel for Content Management

## Installation

1. Clone the repository
2. Run `composer install`
3. Run `npm install` and `npm run dev` or `npm run build`
4. Copy `.env.example` to `.env` and configure your database settings
5. Run `php artisan key:generate`
6. Run `php artisan migrate` to set up the database
7. Run `php artisan serve` to start the local development server

## Technologies
- Laravel
- PHP
- MySQL
- HTML / CSS / JS / Tailwind CSS (if applicable)
